## Evaluation Workflow

For each world passing through validation, complete the following:

1. **Content Ownership Audit:** Review every visible element in the browser. Is it editable? Should it be?
2. **Simulated Client Workflow:** Act as a freelancer and change all the copy to a specific client brief. Do not open VS Code. Stress test the UI with ugly inputs (long titles, empty fields, extreme colors).
3. **Identity Preservation Test:** After completely replacing the copy, step back. Does the world still feel like itself, or did it lose its emotional resonance?
4. **Export Quality Audit:** Export the code and review it. Would you be embarrassed to commit this to GitHub? (Check for dead code, poor prop names, hardcoded orphans).
