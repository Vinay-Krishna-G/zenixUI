# The Experience Lab

This directory is an isolated R&D environment. 

Whenever you think *"What if..."*
- Do **not** put it into Aurora.
- Do **not** put it into Hanami.
- Put it here.

Only after users love an experimental feature should it be promoted to a world. This protects our flagship worlds from becoming volatile playgrounds.

## Structure
- `physics/` - Spring timings, easing functions, scroll momentum
- `typography/` - Layout rhythms, scale experiments, variable fonts
- `motion/` - Complex orchestrated animations (e.g. GSAP/Framer prototypes)
- `materials/` - Glassmorphism, noise, textures, lighting algorithms
- `interactions/` - Hover states, focus rings, complex cursors
